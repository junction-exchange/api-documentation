---
title: 'GraphQL Queries and mutations'
api: 'POST /graphql'
---
Query to fetch all supported tokens:
```
query chainsV2 {
  routingV2 {
    chainsV2 {
      name
      tokens {
        asset {
          id
          name
          symbol
          contract
        }
      }
    }
  }
}
```
