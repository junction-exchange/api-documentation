---
title: "Junction exchange fees"
description: "Discover the fee rates applied on all transactional features"
---

Currently, all fees generated on **Junction Exchange** are converted into **XP**. This applies to all users who connect their wallet in the **Earn** section and works across all ecosystems and address types.

**_Note:_**_ The _**_\$JUN  token (coming soon)_**_ will offer tiered rebates on all transaction fees._

## Swap

| **Transaction nature**      | **Provider Fee %** | **Junction Fee %** |
| --------------------------- | ------------------ | ------------------ |
| Same asset transfer/bridge  | 0                  | 0                  |
| Swap on the same blockchain | 0.10               | 0.10               |
| Swap to another blockchain  | 0.10               | 0.30               |
|                             |                    |                    |
